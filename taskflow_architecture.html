Tech stack & key patterns

Core dependencies

  • Spring Boot 3.2
  • Spring Data JPA
  • Spring Validation
  • Spring Actuator
  • PostgreSQL 16
  • Lombok

Auth & security

  • Identity via X-User-Id header
  • Roles: ADMIN, MANAGER, USER
  • Only creator/assignee can edit
  • Only ADMIN/MANAGER can assign
  • No JWT / session (stateless)

Async & caching

  • AsyncConfig thread pool
  • Core pool: 5, Max: 10
  • Queue capacity: 100
  • In-memory TaskCache
  • Notifications via async

Testing

  • TaskServiceTest (unit)
  • AnalyticsServiceTest (unit)
  • TaskStatusTest (state machine)
  • test_api.sh (e2e shell)
  • Surefire reports included

Task status state machine

TODO → IN_PROGRESS → IN_REVIEW → COMPLETED

CANCELLED  reachable from any non-terminal state  ·  IN_REVIEW can step back to IN_PROGRESS  ·  COMPLETED and CANCELLED are terminal
